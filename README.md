## DEPENDECIES/TECHNOLOGIES
- Framework: **Nestjs (typescript version)**
- Database ORM: **TypeORM**
- Database: **use any of MySQL / MariaDB / Postgres / CockroachDB / SQLite / Microsoft SQL Server / Oracle / sql.js / MongoDB**
- Test Library:
- CI Tool: **Cirlce CI**

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
